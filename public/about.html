About this Website

This website was created using a combination of HTML, CSS, and JavaScript code. The chat functionality was made possible with the use of Node.js, Express.js, and Socket.io. The server is run on a Codio box using Ubuntu 18.04 LTS. 🐧

🏢 Website Design

Styling 🎀

The website has been designed with consistency in mind, with a common use of rounded corners throughout the pages to reflect modern design patterns. Great detail has been taken into ensuring the padding of the website remains consistent on both larger and smaller screens.

The chat page has been carefully designed to make different elements visible and distinguishable. The outgoing and incoming messages are positioned on opposite sides of the page and are of different colours. The message alerts are separated from the main conversation by their placement in the centre of the chat, while the typing notification is a soft grey colour to represent its temporary presence.

Chat Usernames 📛

The chat usernames are a randomly selected string of adjectives, nouns, and emojis, generated by the JavaScript back-end. These memorable phrases make the chat easy to follow and fun to use! However, due to it being randomly generated, I am not responsible for any...unfortunate combination of words.
